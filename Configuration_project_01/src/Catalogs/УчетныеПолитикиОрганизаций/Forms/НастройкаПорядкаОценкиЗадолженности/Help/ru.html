Настройка порядка оценки задолженности

Форма предназначена для настройки порядка оценки задолженности по видам задолженности. Настройка применяется к учетной политике организаций. Настраиваемые виды задолженности:

  • Выданные авансы
  • Задолженность контрагентов в рублях
  • Задолженность контрагентов в валюте и у.е.
  • Займы выданные

Для каждого вида задолженности можно задать один или несколько интервалов оценки:

  • Наименование - формируется автоматически
  • от, дней - количество дней, прошедших с даты возникновения задолженности, с которого применяется оценка
  • Оценка, % - процент оценки задолженности (от 0 до 100)
