Хозяйственные операции

Справочник представляет собой предустановленный список хозяйственных операций, который не может быть изменен пользователем.
Хозяйственная операция - отдельное событие (факт) финансово-хозяйственной деятельности предприятия, зафиксированное первичным учетным документом и вызывающее изменение в составе имущества, обязательств или капитала предприятия, а также источниках их образования.

Движения по хозяйственным операциям служат источником данных для формирования проводок в подсистеме Международный финансовый учет и для получения фактических данных в подсистеме Бюджетирование.

Для формирования в Международном финансовом учете проводок по отраженным в программе хозяйственным операциям необходимо предварительно осуществить настройку Шаблонов проводок, относящихся к выбранным хозяйственным операциям.
