Оформление передач и выкупов

Рабочее место предназначено для оформления документов передачи товаров между организациями и выкупов товаров с хранения по распоряжениям.

Рабочее место с настройками оформления документов передач товаров между организациями открывается из раздела Финансовый результат и контроллинг - Передачи к оформлению или из журнала Документов между организациями по гиперссылке Передачи.

Рабочее место с настройками оформления документов выкупа товаров с хранения открывается из раздела Закупки - Выкупы к оформлению или из журнала Документов закупок по гиперссылке Выкупы.
